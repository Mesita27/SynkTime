Sistema Biométrico SynkTime

Demostración del sistema de reconocimiento biométrico para registro de asistencia

Información: Esta es una demostración del sistema biométrico implementado. El sistema incluye verificación por huella digital usando WebAuthn API y reconocimiento facial usando face-api.js. Para una funcionalidad completa, se requiere HTTPS y dispositivos compatibles.

Huella Digital

Verificación rápida y segura mediante lectores biométricos compatibles con WebAuthn

Reconocimiento Facial

Análisis facial avanzado usando inteligencia artificial en el navegador

Seguridad

Datos encriptados y almacenamiento seguro de templates biométricos

Compatibilidad

Funciona en dispositivos móviles, tablets y computadoras modernas
